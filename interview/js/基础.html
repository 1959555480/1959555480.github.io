# 什么是事件循环

  • 同步和异步任务分别进入不同的执行场所，同步的进入主线程，异步的进入Event Table并且注册函数
  • 当指定的事情完成时，Event Table会将这个函数移入Event Queue
  • 主线程内的任务执行完毕为空，回去Event Queue读取对应的函数，进入主线程执行
  • 上述过程会不断重复，也就时常说的Event Loop
